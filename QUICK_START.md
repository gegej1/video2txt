# 🚀 Video2Txt 快速开始指南

## 一分钟快速上手

### 1️⃣ 启动应用

```bash
# macOS/Linux
./start.sh

# Windows (双击运行或在命令行执行)
start.bat
```

### 2️⃣ 打开浏览器

访问：http://localhost:5000

### 3️⃣ 上传文件

- **拖拽**：直接拖动视频/音频到浏览器窗口
- **点击**：点击上传区域选择文件

### 4️⃣ 选择模型

- **Small**：快速，适合测试
- **Medium**：⭐ 推荐，准确率高
- **Large**：最慢，准确率最高

### 5️⃣ 开始转换

点击"开始转换"按钮，等待处理完成

### 6️⃣ 下载结果

转换完成后，点击"下载文本"保存结果

## 常见问题

### Q: 首次使用很慢？
A: 首次使用会下载模型文件（1-3GB），下载完成后就快了。

### Q: 如何提高准确率？
A: 使用 Large 模型，确保音频清晰。

### Q: 可以批量处理吗？
A: 可以逐个上传处理，或使用命令行批量脚本 `batch_transcribe.py`。

### Q: 支持什么语言？
A: 默认中文，可修改配置支持英文、日文等多种语言。
